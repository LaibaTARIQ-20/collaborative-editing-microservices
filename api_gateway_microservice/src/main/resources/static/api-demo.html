Collaborative Editing System

Spring Boot Microservices Architecture

Similar to Google Docs - Multiple users can edit documents collaboratively with real-time tracking

System Architecture

✅ API Gateway (Port 8084) - Routes requests to appropriate services

✅ Service Discovery (Port 8761) - Eureka Dashboard

✅ Database Integration - MySQL with JPA/Hibernate

✅ JUnit Tests - Automated testing with Mockito
